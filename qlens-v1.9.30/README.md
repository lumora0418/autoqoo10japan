# QLens (큐렌즈) — Qoo10 Japan 셀러 가격·마진 관리 도구

> 한국 상품을 일본 큐텐재팬(Qoo10 Japan)에 파는 셀러를 위한 크롬 확장 프로그램입니다.
> **소싱가만 넣으면 일본 판매가·마진을 자동 계산**하고, QSM 상품을 한 표에서 관리·수정합니다.

---

## 🟢 이 도구가 하는 일 (쉽게)

큐렌즈를 쓰면 이런 게 됩니다.

- 큐텐에 올려둔 **내 상품 전부를 한 표로** 불러옵니다.
- 한국 소싱가(원화)를 넣으면 → **일본 판매가(엔화)와 마진율을 자동으로 계산**해 줍니다.
- **평상시 / 메가포 / 메가와리** 행사별 적정 가격을 한 번에 보여줍니다.
- 가격·재고를 표에서 바로 고치고 → **버튼 한 번으로 큐텐에 반영**합니다.
- 모든 데이터를 **내 구글 스프레드시트에 자동 백업**합니다.
- (선택) AI가 정산 데이터를 보고 **매출 개선점을 조언**합니다.

> 💬 프로그래밍을 몰라도 됩니다. 아래 순서대로 **복사·붙여넣기와 클릭**만 따라 하면 됩니다.
> 처음 설정은 **약 15분**, 한 번만 하면 됩니다.

## 🧭 전체 그림 (한 장 요약)

```
[1] 크롬에 큐렌즈 설치
        │
[2] 구글 스프레드시트 + Apps Script 연결   ← 데이터 저장소 (필수)
        │

        │
[5] 상품 불러오기 → 소싱가 입력 → 가격 반영  ← 매일 쓰는 부분
```

## 1️⃣ 확장 프로그램 설치

1. 받은 **`qlens-v1.9.30.zip`** 파일의 압축을 풉니다 → `qlens-v1.9.30` 폴더가 생깁니다.
2. 크롬 주소창에 **`chrome://extensions`** 입력 후 이동합니다.
3. 화면 **오른쪽 위 "개발자 모드"** 스위치를 **켭니다**.
4. 왼쪽 위 **"압축해제된 확장 프로그램을 로드합니다"** 버튼 클릭 → 방금 푼 **`qlens-v1.9.30` 폴더 선택**.
5. 큐렌즈 아이콘이 생기면 성공입니다. (아이콘이 안 보이면 주소창 오른쪽 🧩 퍼즐 아이콘 → 큐렌즈 옆 📌 압정 클릭)

> 🔄 **나중에 새 버전(zip)을 받으면**: 같은 폴더에 덮어쓴 뒤 `chrome://extensions`에서 큐렌즈의 **새로고침(↻)** 버튼만 누르면 됩니다.

## 2️⃣ 구글 스프레드시트 + Apps Script 연결 (★ 가장 중요)

큐렌즈는 데이터를 **내 구글 스프레드시트**에 저장합니다. 이 연결을 먼저 만들어야 합니다.
어렵게 들리지만, **코드 한 덩어리를 복사해 붙여넣고 버튼 몇 개 누르는 것**이 전부입니다.

### 2-1. 빈 스프레드시트 만들기
1. [sheets.new](https://sheets.new) 접속 (구글 로그인 상태에서) → 빈 시트가 새로 만들어집니다.
2. 시트 이름을 알아보기 쉽게 바꿉니다. (예: `큐렌즈 상품관리`)

### 2-2. Apps Script 편집기 열기
1. 그 시트 상단 메뉴에서 **확장 프로그램 → Apps Script** 클릭.
2. 새 탭에 코드 편집기가 열립니다. 안에 있는 `function myFunction() {}` 같은 **기존 내용은 전부 지웁니다.**

### 2-3. 큐렌즈 코드 붙여넣기
1. 큐렌즈 폴더 안의 **`QSM_Lens_GAS_v1.8.4.gs`** 파일을 메모장(또는 텍스트 편집기)으로 엽니다.
2. **전체 선택(Ctrl+A) → 복사(Ctrl+C)**.
3. 2-2의 Apps Script 편집기에 **붙여넣기(Ctrl+V)**.
4. **★ 본인 스프레드시트 ID 넣기 (꼭!)** — 코드 위쪽의
   ```
   const SHEET_ID = 'PASTE_YOUR_SPREADSHEET_ID_HERE';
   ```
   줄에서 작은따옴표 안을 **내 시트 ID로 교체**합니다.
   - 내 시트 ID는 시트 주소에서: `docs.google.com/spreadsheets/d/`**`[이 부분]`**`/edit`
   - 예) 주소가 `.../d/1AbcD123.../edit` 이면 → `const SHEET_ID = '1AbcD123...';`
   - (이걸 안 바꾸면 "SHEET_ID 오류"가 나며 작동하지 않습니다.)
5. 위쪽 **💾 저장(디스크 아이콘)** 클릭.

### 2-4. 웹 앱으로 배포하기 (외부에서 부를 수 있게)
1. 편집기 오른쪽 위 파란색 **배포 → 새 배포** 클릭.
2. "유형 선택"의 ⚙️(톱니) → **웹 앱** 선택.
3. 아래처럼 설정합니다. **이 두 줄이 가장 중요합니다.**
   - **실행 계정(다음 사용자로 실행):** `나`
   - **액세스 권한(액세스 권한이 있는 사용자):** **`모든 사용자`** ← 반드시 이걸로!
4. **배포** 클릭 → 구글 권한 승인 창이 뜨면 **내 계정 선택 → "고급" → "(안전하지 않음) 이동" → 허용**.
   (내가 만든 내 스크립트라 안전합니다. 구글이 미검토 앱이라 경고를 띄우는 것뿐입니다.)
5. 마지막에 나오는 **"웹 앱 URL"** (`https://script.google.com/macros/s/.../exec`)을 **복사**합니다.

### 2-5. URL을 큐렌즈에 입력
1. 크롬에서 큐렌즈 아이콘 클릭 → **⚙️ 설정** 열기.
2. **📊 스프레드시트 (Apps Script) 연동** 칸에 방금 복사한 **웹 앱 URL 붙여넣기**.
3. **연결 테스트** → ✅ 표시가 나오면 성공 → **저장**.

### 2-6. 시트 자동 만들기
- 상품관리 화면에서 **시트 자동 세팅** 기능을 한 번 실행하면, 필요한 탭(`QSM_Lens_Items`, `QSM_Lens_Config` 등)이 자동으로 생성됩니다.

> ⚠️ **나중에 큐렌즈 코드(.gs)가 업데이트되면** — 새 코드를 붙여넣은 뒤,
> **배포 → 배포 관리 → (기존 배포의) ✏️ 연필 → 버전 "새 버전" → 배포** 순서로 하세요.
> "새 배포"로 하면 **URL이 새로 바뀌어** 연결이 끊깁니다. 반드시 **기존 배포를 편집**해야 URL이 유지됩니다.

## 3️⃣ 큐텐 QSM API 키 발급·입력 (내 상품 불러오기용)

큐렌즈가 내 큐텐 상품 목록·가격·재고를 가져오려면 **QSM API 인증키**가 필요합니다.

1. [큐텐 QSM(셀러센터)](https://qsm.qoo10.jp) 에 로그인합니다.
2. **왼쪽 메뉴 맨 아래의 `QAPI 개발 가이드`** 를 클릭합니다.
3. 화면 중간의 **`Qoo10 API Key` → `API ID 발급/조회`** 에서 **인증키(GiosisCertificationKey)** 를 발급하고 복사합니다.
4. 큐렌즈 **⚙️ 설정 → 🔑 Qoo10 QSM API 연동** 에 입력:
   - **API Key**: 위에서 받은 인증키
   - **판매자 ID**: QSM 로그인 아이디(이메일)
   - **판매자 비밀번호**: QSM 로그인 비밀번호
5. **저장**.

> 🔒 입력한 키·비밀번호는 **내 브라우저 안에만 암호화되어 저장**되며 외부로 전송되지 않습니다.
> 인증키 발급 위치를 못 찾으면 큐텐 셀러 고객센터에 "API 인증키(GiosisCertificationKey) 발급" 문의하면 됩니다.

## 4️⃣ AI 매출분석 키 발급 (선택 — 안 넣어도 됩니다)

정산 데이터를 AI가 분석해 조언해 주는 기능입니다. **하나만 넣어도 되고, 안 넣어도 나머지 기능은 다 됩니다.**
무료부터 시작하길 권합니다.

| AI | 비용 | 발급 사이트 | 키 모양 |
|---|---|---|---|
| **Gemini** | 🆓 무료 (추천) | [aistudio.google.com/apikey](https://aistudio.google.com/apikey) | `AIza...` |
| **Groq** | 🆓 무료 | [console.groq.com/keys](https://console.groq.com/keys) | `gsk_...` |
| **Claude** | 💳 유료 | [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys) | `sk-ant-...` |
| **ChatGPT** | 💳 유료 | [platform.openai.com/api-keys](https://platform.openai.com/api-keys) | `sk-...` |

**발급 방법 (공통):**
1. 위 표의 사이트에 접속 → 구글/이메일로 로그인(회원가입).
2. **"Create API Key"** (키 만들기) 버튼 클릭 → 생성된 키를 **복사**.
3. 큐렌즈 **⚙️ 설정 → 🤖 AI 매출 분석** 에서 해당 칸에 붙여넣기.
4. 위쪽 **"사용할 AI"** 에서 사용할 것을 선택 → **저장**.
   - 선택한 AI 키가 비어 있으면, 입력해 둔 다른 키로 **자동으로 대신 사용**합니다.
5. (선택) **셀러 컨텍스트 프롬프트** 칸에 내 판매 상황을 적어두면, AI가 항상 그걸 기반으로 조언합니다.

> 💡 무료 Gemini만 넣어도 충분합니다. 더 정밀한 조언을 원하면 Claude(유료)를 추가하세요.

## 5️⃣ 첫 사용 — 상품 불러오기부터 가격 반영까지

설정이 끝났으면 실제로 써 봅니다.

1. 큐렌즈 아이콘 → **"📦 상품 관리 열기"**.
2. **[📦 QSM 상품 불러오기]** 클릭 → 내 큐텐 상품이 표로 들어옵니다. (상품 많으면 30초~1분)
   - 상태 칸에 **재고(📦 숫자)** 와 **판매/품절/중지/단종** 배지가 표시됩니다.
3. 각 상품의 **소싱가(₩)** 칸에 한국 구매가를 입력합니다.
   - 그러면 **총비용·추천 판매가(¥)·마진율**이 자동 계산됩니다.
   - 무게(kg)를 넣으면 국제배송비가 자동 반영됩니다.
4. 가격이 마음에 들면, 맨 오른쪽 **작업** 칸의 **● 적용** 버튼으로 큐텐에 가격을 보냅니다.
5. 작업이 끝나면 헤더의 **[💾 저장+QSM반영]** 또는 **[📤 시트에 저장]** 으로 변경사항을 시트에 저장합니다.

> ✅ 한 번 저장해 두면, 다음부터는 **[📥 시트 불러오기]** 로 1~2초 만에 내 데이터를 다시 불러옵니다.

## 📅 매일 쓰는 순서 (요약)

```
① 📥 시트 불러오기 (저장해 둔 데이터 로드)
② 소싱가/무게 등 수정
③ ● 적용 (필요한 상품만 큐텐에 가격 반영)
④ 💾 저장+QSM반영 (변경사항 시트에 저장)
```

- **품절 처리**: 작업 ▼ → `품절 처리(재고 0)` / 다시 팔려면 `재입고`
- **재고 수정**: 상태 칸의 `📦 숫자` 칩 클릭 → 수량 입력
- **소싱 페이지 가격 확인**: 소싱처 칸의 **↗** 버튼 → 소싱페이지가 **같은 창의 새 탭**으로 열리고 사이드패널이 함께 열림 → 새 탭을 **우클릭 → "분할 보기"** 로 큐렌즈와 좌우로 나란히 보면 편함 → (확인창 거쳐) 시트 반영

## 💡 주요 기능

### 🔍 상품 관리
- 소싱가(₩)+무게(kg) → **추천판매가(¥)·마진율 자동 계산**, 평상시·메가포·메가와리 동시 표시
- 마진율 색상 구분: 🔴 역마진 / 🟡 저마진 / 🟢 고마진
- 표에서 바로 수정(인라인 편집), **Ctrl+Z 되돌리기**
- 검색(브랜드/상품명/코드/URL) + 필터(상태·소싱처·브랜드·마진·단품/세트)

### 📦 재고·상태
- QSM 재고 표시, **재고 0 = 품절** 자동 구분, 칩 클릭으로 즉시 수정
- 품절/단종/재입고/활성화 처리

### 📦 기획세트(번들)
- 세트 1개 = 구성품 여러 개를 1:N로 관리, 소싱가·무게 **자동 합산**(증정 제외)
- 단품 ↔ 세트 양방향 전환

### 🔗 소싱 모니터(사이드패널)
- 지원: 올리브영 · 네이버(스마트스토어/브랜드/쇼핑) · 쿠팡 · 무신사 · 마켓컬리 · 다이소 · Cafe24(더메디큐브/VT/아누아)
- 현재가 추적 → 확인 후 시트 반영

### 🤖 AI 매출분석
- 정산 데이터 기반 매출 패턴·개선점 조언 (Gemini/Groq/Claude/ChatGPT)

### 📤 시트 동기화
- `📖 사용설명서`(맨 앞 안내) · `QSM_Lens_Items`(메인) · `QSM_Lens_Bundles`(세트) · `QSM_Lens_ShipRates`(배송요율) · `QSM_Lens_Config`(설정) 자동 생성·동기화
- 🆕 **마진계산기 데이터를 직접 옮기기**: `QSM_Lens_Items` 탭의 입력 컬럼(소싱가 G·무게 H·마진율 K 등)에 **값만 붙여넣기**한 뒤 확장에서 **[🔧 시트 자동 세팅]**을 누르면 수식이 채워져 자동 계산됩니다. (자세한 컬럼 안내는 스프레드시트 `📖 사용설명서` 시트 참고)

## ❓ 자주 묻는 질문

### Q. "시트 webhook 미연결" 이라고 떠요
- **2️⃣ 단계**의 웹 앱 URL이 설정에 안 들어갔거나, 배포 시 **액세스 권한이 "모든 사용자"가 아닐** 때 납니다.
- 설정에서 URL을 다시 붙여넣고 **연결 테스트** → 안 되면 Apps Script 배포의 액세스 권한을 `모든 사용자`로 다시 배포하세요.

### Q. 상품 불러오기가 로그인 화면(HTML)을 반환해요 / 안 돼요
- 거의 항상 **배포 액세스 권한** 문제입니다. `모든 사용자`로 재배포하세요.
- QSM API 키/ID/비밀번호 오타도 확인하세요.

### Q. QSM API 인증키를 어디서 발급하나요?
- 큐텐 QSM(셀러센터) 로그인 → **왼쪽 메뉴 맨 아래 `QAPI 개발 가이드`** → 중간의 **`Qoo10 API Key` → `API ID 발급/조회`** 에서 **인증키(GiosisCertificationKey)** 발급.

### Q. AI 키는 꼭 넣어야 하나요?
- 아니요. **선택**입니다. 안 넣어도 상품관리·가격계산·시트저장은 다 됩니다. 무료 Gemini만 넣어도 분석이 됩니다.

### Q. 상품 불러오기가 너무 오래 걸려요
- 100개 초과면 자동 동기화 OFF. 헤더 **📮 배송비 동기화**를 수동 실행하세요. (재고도 이때 함께 채워집니다)
- 한 번 불러오면 시트에 저장되니, 이후엔 **📥 시트 불러오기**로 빠르게.

### Q. 소싱가가 자꾸 이상한 값(예: 49500)으로 바뀌어요
- 과거 버전의 **소싱 데이터 가져오기 오류**였고 v1.9.30에서 고쳤습니다. 잘못 들어간 값은 소싱가 칸에서 올바르게 고친 뒤 저장하세요.
- 사이드패널 가격 반영 시엔 이제 **확인창**이 뜨니, 값이 이상하면 취소하세요.

### Q. 사이드패널이 안 열려요
- 크롬 114 이상 필요. 안 되면 새 탭으로 열립니다.

### Q. 화면 마진율과 스프레드시트 마진율이 달라요
- v1.9.30에서 일치하도록 수정했습니다. **GAS 코드를 v1.8.4로 재배포**하고, 해당 상품을 다시 저장하면 같아집니다.
