📘 Google Apps Script 설정 가이드
QLens는 Apps Script 파일 전체를 교체하는 방식으로 설정합니다.
기존 코드에 추가하거나 편집할 필요 없이, 아래 파일을 그대로 붙여넣으면 됩니다.
  1. QLens 설정에서 Q10Auto_QSMLens_v1.6.1.gs 파일을 받습니다
    QLens 설정 → 스프레드시트 섹션에서 다운로드
  2. Google Sheets 열기 → 확장 프로그램 → Apps Script
  3. 에디터에서 기존 코드 전체를 선택(Ctrl+A) 후 삭제
  4. 다운로드한 .gs 파일 내용 전체를 붙여넣기 → 저장(Ctrl+S)
  5. 처음 설치: 상단 배포 → 새 배포 → 유형 웹 앱 → 액세스 모든 사용자 → 배포 → URL 복사
  6. QLens 설정 → webhook URL에 붙여넣기 → 저장
  7. 상품관리 탭으로 돌아와 🔧 시트 자동 세팅으로 시트를 만든 뒤, 🔄 전체 자동 동기화 한 번이면 상품·소싱·브랜드·배송비가 한꺼번에 채워집니다
⚠️ 코드를 수정/업데이트한 경우 — 반드시 "재배포"
코드를 붙여넣고 저장(Ctrl+S)만 하면 반영되지 않습니다. 기존 /exec 주소는 옛 버전을 계속 실행합니다.
→ 배포 → 배포 관리 → 연필(편집) → 버전: "새 버전" 선택 → 배포
URL은 그대로 유지됩니다. "새 버전"으로 배포해야 최신 코드가 적용됩니다.
🩺 데이터가 안 읽힐 때 점검 순서
① 위 "새 버전 재배포"를 했는가 (가장 흔한 원인)
② webhook URL이 /exec로 끝나는가 (끝이 /dev면 안 됨)
③ Apps Script의 SHEET_ID가 현재 시트 URL의 ID와 같은가
④ 시트 탭 이름이 QSM_Lens_Items / QSM_Lens_Sourcing인가
