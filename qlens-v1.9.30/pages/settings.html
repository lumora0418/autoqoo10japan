🤖 AI 매출 분석
정산 CSV 분석 시 AI가 매출 패턴·개선점을 조언합니다. 키는 하나만 넣어도 되고, 여러 개 넣으면 위에서 선택한 AI를 우선 사용합니다. (선택한 키가 비어 있으면 입력된 다른 키로 자동 폴백)
💡 Gemini 키: aistudio.google.com/apikey (무료)
Groq 키: console.groq.com/keys (무료)
Claude 키: console.anthropic.com/settings/keys (유료)
ChatGPT 키: platform.openai.com/api-keys (유료)
입력해두면 AI 조언 시 항상 이 내용을 먼저 읽고 대답합니다. 자유롭게 작성하세요.

📊 스프레드시트 (Apps Script) 연동
Google Sheets와 연결해서 상품·키워드 데이터를 영구 보관합니다. 한 번만 설정하면 됩니다.
📥 .gs 파일 다운로드
💡 처음 설정 순서:
① 📥 .gs 파일 다운로드 → Google Sheets 만들기
② Sheets → 확장 프로그램 → Apps Script → 코드 전체 삭제 후 붙여넣기
③ 저장 → 배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자 → 배포
④ 받은 URL을 위 입력칸에 붙여넣고 저장
⑤ 🧪 연결 테스트 → ✅ 표시 확인 → 상품관리 탭에서 🔧 시트 자동 세팅
📅 기본 데이터 보존 기간
스냅샷 데이터 보존 기간을 설정합니다. 대시보드에서도 변경 가능합니다.
🛍️ 신규주문 알림
